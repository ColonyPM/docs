# list

Lists all Deployed packages.

## Usage

```bash
$ cpm deploy list
```

### Aliases

- `ls`

## Examples

```bash
$ cpm deploy list
| Package  | Name         | Container ID | Created At           |
|:---------|:-------------|:-------------|:---------------------|
| fibbers  | exec-a       | abc123       | 2026-01-01T12:00:00Z |
| fibbers  | exec-b       | def456       | 2026-01-01T12:00:00Z |
| primes   | No Executors |              | 2026-01-02T08:30:00Z |
```
