# install

Downloads a package from the [Colony PM registry](https://colonypm.xyz).

## Usage

```bash
cpm pkg install <package-name>
```

## Arguments

| Argument | Description | Required |
|----------|-------------|----------|
| `<PACKAGE-NAME>` | Name of the package to install | Yes |


## Examples

Install a package named `faas`:

```bash
cpm install faas
```
