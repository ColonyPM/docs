# pkg

`cpm pkg` runs package management commands - working with local packages and the Colony PM registry.


## Commands

- `cpm pkg init [DEST]`
- `cpm pkg install <PACKAGE-NAME>`
- `cpm pkg upload [PATH] --token <TOKEN>>`
- `cpm pkg remove <PACKAGE-NAME>`
- `cpm pkg list`
- `cpm pkg values [DEST]`
- `cpm pkg get <PACKAGE-NAME>`

### When not to Use

Commands related to colony management is wraped with `cpm deploy`.
